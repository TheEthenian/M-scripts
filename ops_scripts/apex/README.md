
==========================================
VARIANTS
==========================================
Apex-main.py -> deals full export from caido
Apex-singular.py -> deals findings json from caido


==========================================
Purpose
==========================================
> parse folders containing json http files to find specific keywords


==========================================
Features
==========================================
> only works with specified wildcard hosts to clear the noise

> returns text extract of the keywords specified depending on the word radius specified, also the http file details and various metadata for easy location in caido

==========================================
Actionable Variables
==========================================
> hostname
> api path
> http method
> keyword



===========================================
Instructions
===========================================

-> Place your json file in the json-file directory
-> Configure the search parameters in the search_config/search.json file.
